# watchfridge
The product for fridge, able to watch in the fridge any where.
冷蔵庫のためのプロダクトです。冷蔵庫の中身をどこにいても見ることができるシステムです。

# 動作環境
## Hardware (client)
* iPhone or Android
## Hardware (camera)
* Raspberry Pi Zero 2 W
* CSI Camera module
* 5V ring light
* Need Network
* FTP server
## Software (camera)
* Raspberry Pi OS Lite
* OpenCV 3.4.3
* numpy 1.21.0
